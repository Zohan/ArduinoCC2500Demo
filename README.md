ArduinoCC2500Demo
=================

A demo of the Arduino communicating with a CC2500.
This uses the minimum amount of pins on the most minimal amount of hardware needed to get the system functioning.

Pinouts CC2500->Arduino

MISO -> 12

MOSI -> 11

SCLK/SCK -> 13

CSN/SS - > 10

VCC -> 3.3v

GND - > GND
